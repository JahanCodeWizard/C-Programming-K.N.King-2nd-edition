# Boolean is C
since C programming language didn't include boolean types in the beginning, but the need grew over time, in the c99 standard the `_Bool` has been introduced. 
This part is actually inspired by this book from chapter 5 pages 84 to 86.

This directory contain some programming files with C extension format, which is intended to show how programmers work with booleans in C89 and C99. 

There may be other methods which may no present here.
